3> what is manage.py in django
ans->manage.py ek Python script hota hai jo Django project ko manage karne ke liye use hota hai.

Kya-kya kar sakte ho:
    Server run kar sakte ho: python manage.py runserver
    App create karna
    Migrations, superuser banana, shell open karna, etc.

 Yeh basically django-admin ka shortcut hota hai with project-specific settings.
